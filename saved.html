Hangman the Guessing Game
The Hangman Game is about guessing letters in a random word. The player has 15 tries to succeed or fail. Wins and losses are recorded on the page.

-The game uses Basic Javascript
-The game uses External Stylesheets: CSS and BootStrap
-The possible words are in an array; the word is chosen randomly using Math.random
-Each new game resets the guesses, the target word and the guess count
-The game uses document.onkeyup to detect a key being pressed
-When a letter is pressed, a for loop cycles through the target word's letters to look for a match
-The text of the game is put on the page with: document.querySelector

Floating Spaceship - Physics

The Spaceship is defined as a Car object that holds: this.position, this.velocity, this.acceleration, this.topspeed, this.xoff, this.yoff, this.r and this.accVector
Car.prototype.update updates the position with velocity (limited speed) and acceleration
Car.prototype.applyForce updates the acceleration with force
Car.prototype.turnLeft and Car.prototype.turnRight use .rotate
Car.prototype.display draws the car and the head of the car is determined by this.accVector.heading()
Car.prototype.checkEdges automatically transports the car to the other side of the screen when it reaches one of the edges
keyPressed can process key strokes: LEFT, RIGHT and the letter z (keyCode 90)
draw updates the car's position as time goes by
